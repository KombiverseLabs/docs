# Available StackKits

## base-homelab

**Single-server homelab with Docker and Dokploy**

- **Nodes:** 1 main node
- **Platform:** Docker
- **Services:** Traefik, Dokploy, Portainer
- **Best for:** Beginners, single-server setups

## modern-homelab

**Multi-node Docker Swarm with Coolify**

- **Nodes:** 2+ nodes
- **Platform:** Docker Swarm
- **Services:** Traefik, Coolify, monitoring
- **Best for:** Production homelabs

## ha-homelab

**Kubernetes high availability cluster**

- **Nodes:** 3+ nodes
- **Platform:** k3s
- **Services:** Ingress, storage, monitoring
- **Best for:** Enterprise-grade setups
