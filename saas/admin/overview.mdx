# KombiSphere Admin

KombiSphere Admin is the **management platform** for tool evaluation, pattern management, and StackKit configuration.

## Key features

- Tool and pattern evaluation
- StackKit management
- Category-based organization
- Discovery engine (GitHub, npm, PyPI)
- Version tracking

## Tech stack

- **Backend:** Go + PocketBase
- **Frontend:** SvelteKit
- **License:** BSL-1.1
