Deploy kombify Cloud on Kubernetes for production.

## Helm chart

```bash
# Add repository
helm repo add kombify https://charts.kombify.io

# Install
helm install kombify kombify/kombify \
  --namespace kombify \
  --create-namespace
```

## Manual deployment

```yaml
# kombistack-deployment.yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: kombistack
spec:
  replicas: 1
  selector:
    matchLabels:
      app: kombistack
  template:
    metadata:
      labels:
        app: kombistack
    spec:
      containers:
      - name: kombistack
        image: ghcr.io/kombiverselabs/kombistack:latest
        ports:
        - containerPort: 5260
```
