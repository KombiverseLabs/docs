kombify Cloud combines open-source tools with a SaaS management layer.

## Components

### Open core tools
- **KombiStack:** Control plane
- **KombiSim:** Simulation engine
- **StackKits:** Infrastructure blueprints

### SaaS platform
- **kombify Cloud Cloud:** User portal
- **kombify Cloud Admin:** Admin center
- **kombify Cloud API:** API gateway

## Data flow

User → Cloud Portal → API Gateway → Tools
