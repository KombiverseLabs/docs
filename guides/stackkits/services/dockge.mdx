Dockge is an optional Docker Compose management surface at `dockge.<domain>`.

## Prerequisites

<Note>
  Use Dockge for compose-stack operations you intentionally want to manage outside generated StackKits artifacts.
</Note>

## Steps

<Steps>
  <Step title="Open Dockge">
    Open `https://dockge.<domain>` from the Node Hub.
  </Step>
  <Step title="Create an admin account">
    Complete first-run setup and store recovery information outside the server.
  </Step>
  <Step title="Add a stack carefully">
    Keep generated StackKits resources separate from manually managed compose stacks.
  </Step>
</Steps>

## Verify

- Dockge can list and start only the stacks it should manage.
- StackKits-generated services remain managed by StackKits.

<AccordionGroup>
  <Accordion title="A manually edited stack conflicts with StackKits">
