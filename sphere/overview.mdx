# kombify Sphere

**Sphere** is kombify's managed SaaS platform. It provides hosted infrastructure management with enterprise features like SSO, billing, and team collaboration.

## Connecting Your Homelab

The Sphere Agent establishes a secure connection from your homelab to the Sphere cloud:

```mermaid
sequenceDiagram
    participant Homelab
    participant Agent as Sphere Agent
    participant Sphere as Sphere Cloud
    
    Note over Agent,Sphere: Initial Setup
    Agent->>Sphere: Register with pairing token
    Sphere->>Agent: Issue mTLS certificate
    
    Note over Agent,Sphere: Normal Operation
    loop Every 30s
        Agent->>Sphere: Heartbeat + metrics
    end
    
    Sphere->>Agent: Deploy command
    Agent->>Homelab: Execute deployment
    Agent->>Sphere: Report status
```

### Agent Installation

<Steps>
  <Step title="Get pairing token">
    In Sphere dashboard, go to **Homelabs → Add Homelab** and copy your pairing token.
  </Step>
  
  <Step title="Install agent">
    ```bash
    curl -fsSL https://get.kombify.dev/sphere-agent | bash
    ```
  </Step>
  
  <Step title="Configure agent">
    ```bash
    sphere-agent init --token YOUR_PAIRING_TOKEN
    ```
  </Step>
  
  <Step title="Start agent">
    ```bash
    sudo systemctl enable --now sphere-agent
    ```
  </Step>
</Steps>

### SSO Configuration

Sphere uses Zitadel for enterprise-grade authentication:

```mermaid
sequenceDiagram
    participant User
    participant Sphere
    participant Zitadel
    participant IdP as Your IdP (Optional)
    
    User->>Sphere: Access dashboard
    Sphere->>Zitadel: Redirect to login
    
    alt Native Login
        Zitadel->>User: Show login form
        User->>Zitadel: Enter credentials
    else Enterprise SSO
        Zitadel->>IdP: Redirect to corporate IdP
        User->>IdP: Authenticate
        IdP->>Zitadel: Return assertion
    end
    
    Zitadel->>Sphere: Return tokens
    Sphere->>User: Authenticated session
```

Supported identity providers:
- **OIDC**: Any OIDC-compliant provider
- **SAML**: Okta, Azure AD, OneLogin, etc.
- **LDAP**: Active Directory, OpenLDAP
- **Social**: GitHub, Google, Microsoft
