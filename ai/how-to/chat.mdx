kombify AI offers multiple chat interfaces. This guide covers how to use each one effectively.

## Chat widget

The chat widget is a floating bubble embedded in kombify Cloud. Click it to start a conversation without leaving your current page.

**Features:**
- Quick questions while working in the Cloud dashboard
- Context-aware — knows which tool you are currently viewing
- Minimizes to a small bubble when not in use

## Web chat

The full-featured web chat at `chat.kombify.io` provides:

- **Session history** — Browse and resume past conversations
- **Model picker** — Choose which AI model to use
- **Sidebar navigation** — Organize conversations by topic
- **Rich responses** — Code blocks, diagrams, citations from documentation

## Tips for better results

<Steps>
  <Step title="Be specific about your setup">
    Instead of "How do I set up SSL?", try "How do I configure Traefik with Let's Encrypt for my homelab at home.example.com?"
  </Step>
  <Step title="Ask follow-up questions">
    kombify AI remembers the conversation context. You can ask "What about for Kubernetes?" to refine a previous answer.
  </Step>
  <Step title="Use infrastructure context">
    When connected to a running kombify TechStack instance, the AI automatically knows your node status, services, and configuration.
  </Step>
</Steps>
