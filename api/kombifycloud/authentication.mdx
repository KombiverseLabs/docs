## User authentication

Users authenticate through Auth0 Universal Login and SSO at `https://login.kombify.io/`. The Cloudflare API Gateway accepts Auth0 access tokens with:

- `iss`: `https://login.kombify.io/`
- `aud`: `https://api.kombify.io`

Send the token on API requests:

```bash
curl -H "Authorization: Bearer $TOKEN" \
  https://api.kombify.io/v1/me
```

## Machine-to-machine integrations

Approved service integrations use Auth0 client credentials:

```bash
TOKEN=$(curl -s -X POST https://login.kombify.io/oauth/token \
  -H "Content-Type: application/json" \
  -d '{
    "grant_type": "client_credentials",
    "client_id": "YOUR_CLIENT_ID",
    "client_secret": "YOUR_CLIENT_SECRET",
    "audience": "https://api.kombify.io"
  }' | jq -r '.access_token')
```

## Validate token

Validation happens at `api.kombify.io`. Origins receive signed identity headers from the edge and do not receive the original `Authorization` header.

```bash
curl -H "Authorization: Bearer $TOKEN" \
  https://api.kombify.io/v1/me
```
