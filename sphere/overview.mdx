# kombify Sphere

kombify Sphere is the managed SaaS platform that brings all kombify tools together with enterprise features like SSO, billing, and team collaboration.

<Card title="What is kombify Sphere?" icon="circle-info">
  kombify Sphere is the customer-facing portal that provides a unified dashboard, subscription management, SSO gateway, and tool launcher for kombify Stack, Sim, and other tools.
</Card>

## Core capabilities

### User management

- **Profile management** - Avatar, name, email, settings
- **Activity logs** - Audit trail for user actions
- **Team management** - Invite and manage team members
- **Role-based access** - User, Manager, Admin roles

### Authentication

- **Single Sign-On** - Zitadel OIDC integration
- **API tokens** - Generate tokens for API access
- **SSO to tools** - One-click access to Stack, Sim, etc.
- **Session management** - Secure session handling

### Billing

- **Subscription plans** - Free, Pro, Enterprise
- **Stripe integration** - Secure payment processing
- **Invoice management** - Download and view invoices
- **Usage tracking** - Monitor API calls and resources

### Tool launcher

- **kombify Stack** - Launch with SSO
- **kombify Sim** - Launch with SSO
- **Admin Center** - For administrators
- **Custom tools** - Add your own tools

## Architecture

### Tech stack

| Component | Technology | Purpose |
|-----------|------------|---------|
| **Frontend** | SvelteKit 2.x | Modern reactive UI |
| **Database** | PostgreSQL | User data and state |
| **Auth** | Zitadel | OIDC provider |
| **Billing** | Stripe | Payment processing |
| **API Gateway** | Kong OSS | Unified API entry |

### Database schema

```typescript
// Core entities
interface User {
  id: string;
  email: string;
  name?: string;
  avatarUrl?: string;
  role: UserRole;
}

interface Subscription {
  id: string;
  userId: string;
  plan: Plan;
  status: SubscriptionStatus;
  currentPeriodEnd: Date;
}

interface ActivityLog {
  id: string;
  userId: string;
  action: string;
  resource: string;
  createdAt: Date;
}
```
