title: Tool launcher
description: Access KombiStack and KombiSim with SSO
---


Launch tools with one-click SSO authentication.

## How it works

<Steps>
  <Step title="Click tool tile">
    Select KombiStack or KombiSim from dashboard
  </Step>
  <Step title="SSO authentication">
    Automatic token generation and validation
  </Step>
  <Step title="Redirect to tool">
    Seamless access to tool dashboard
  </Step>
</Steps>

## Available tools

- **KombiStack:** Infrastructure control plane
- **KombiSim:** Simulation engine
