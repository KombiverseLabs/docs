## Key features

<AccordionGroup>
  <Accordion title="Single sign-on" icon="key">
    OIDC authentication with Zitadel:
    
    - One account for all tools
    - Enterprise SSO support
    - Multi-factor authentication
    - Session management
  </Accordion>

  <Accordion title="Subscription management" icon="credit-card">
    Stripe-powered billing:
    
    - Free, Pro, Enterprise tiers
    - Usage-based pricing
    - Team management
    - Invoice history
  </Accordion>

  <Accordion title="Tool launcher" icon="rocket">
    Unified access to all tools:
    
    - KombiStack instances
    - KombiSim environments
    - One-click launch with SSO
    - Resource monitoring
  </Accordion>

  <Accordion title="User dashboard" icon="chart-line">
    Centralized management:
    
    - Usage analytics
    - Resource quotas
    - Team collaboration
    - Audit logs
  </Accordion>
</AccordionGroup>

## Tech stack

| Component | Technology | Purpose |
|-----------|------------|---------|
| **Frontend** | SvelteKit 5 | User portal |
| **Backend** | Prisma + PostgreSQL | Data management |
| **Auth** | Zitadel | OIDC provider |
| **Billing** | Stripe | Subscription management |
