# Installation

Install KombiSim using Docker Compose, Docker, or from source.

## Prerequisites

<CardGroup cols={3}>
  <Card title="Docker" icon="docker">
    Docker Engine 24.0+
  </Card>
  <Card title="System resources" icon="microchip">
    2GB RAM, 5GB disk
  </Card>
  <Card title="Ports" icon="network-wired">
    5270, 5271, 2222-2322
  </Card>
</CardGroup>

## Installation methods

<Tabs>
  <Tab title="Docker Compose">
    ```bash
    git clone https://github.com/KombiverseLabs/KombiSim
    cd KombiSim
    docker compose up -d
    ```
  </Tab>

  <Tab title="Docker">
    ```bash
    docker run -d \
      --name kombisim \
      -p 5270:5270 \
      -v /var/run/docker.sock:/var/run/docker.sock \
      ghcr.io/kombiverselabs/kombisim:latest
    ```
  </Tab>

  <Tab title="From source">
    ```bash
    git clone https://github.com/KombiverseLabs/KombiSim
    cd KombiSim
    go build -o kombisim ./cmd/kombisim
    ./kombisim serve
    ```
  </Tab>
</Tabs>
