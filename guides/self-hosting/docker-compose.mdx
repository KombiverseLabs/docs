Deploy the complete kombify Cloud stack using Docker Compose.

## Quick start

```bash
# Clone KombiStack
git clone https://github.com/KombiverseLabs/KombiStack
cd KombiStack

# Start services
docker compose up -d

# Access
# KombiStack: http://localhost:5261
# KombiSim: http://localhost:5271
```

## Services

- KombiStack Core (API + PocketBase)
- KombiStack UI
- KombiSim API
- KombiSim UI
- Headscale (optional)
