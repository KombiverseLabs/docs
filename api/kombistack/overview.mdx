# KombiStack API

KombiStack provides a comprehensive REST API for managing infrastructure stacks, nodes, and deployments.

**Base URL:** `http://localhost:5260/api/v1`

**Authentication:** PocketBase JWT tokens or API keys

## Quick start

### Authentication

```bash
# Login to get JWT token
curl -X POST http://localhost:5260/api/collections/users/auth-with-password \
  -H "Content-Type: application/json" \
  -d '{
    "identity": "admin@kombistack.local",
    "password": "your-password"
  }'

# Use token in subsequent requests
export TOKEN="your-jwt-token"
curl -H "Authorization: Bearer $TOKEN" \
  http://localhost:5260/api/v1/stacks
```

### Create a stack

```bash
curl -X POST http://localhost:5260/api/v1/stacks \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d @kombination.yaml
```

## API structure

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/v1/health` | GET | Health check |
| `/api/v1/stacks` | GET, POST | List/create stacks |
| `/api/v1/stacks/{id}` | GET, PUT, DELETE | Manage specific stack |
| `/api/v1/workers` | GET, POST | List/register workers |
| `/api/v1/jobs` | GET | List jobs |
| `/api/v1/unifier/validate` | POST | Validate configuration |
| `/api/v1/unifier/execute` | POST | Execute unifier pipeline |
