# kombify Stack quick start

Deploy your first infrastructure stack in under 5 minutes.

## Prerequisites

- Docker Engine 24.0+ with Compose v2.20+
- A Linux server (local or VPS) with SSH access
- 4GB RAM, 10GB disk space

## Step 1: Install kombify Stack

```bash
mkdir ~/kombify && cd ~/kombify
curl -O https://raw.githubusercontent.com/soulcreek/KombiStack/main/docker-compose.yml
docker compose up -d
```

Access the dashboard at [http://localhost:5261](http://localhost:5261).

## Step 2: Create your first stack

<Steps>
  <Step title="Start wizard">
    Click **"New Homelab"** â†’ **"Easy Wizard"**
  </Step>

  <Step title="Configure stack">
    ```yaml
    stackkit: base-homelab
    version: "1.0.0"
    
    meta:
      name: "My First Stack"
    
    nodes:
      - name: main-server
        type: local
        connection:
          host: 192.168.1.100
          user: root
    
    services:
      - traefik
      - homepage
      - uptime-kuma
    ```
  </Step>

  <Step title="Install agent">
    On your target server:
    
    ```bash
    curl -sSL https://install.kombify.io/agent | sudo sh -s -- \
      --token YOUR_TOKEN \
      --server http://your-stack-ip:5260
    ```
  </Step>

  <Step title="Deploy">
    Click **"Apply"** and watch the deployment progress.
  </Step>
</Steps>

## Step 3: Access your services

| Service | URL | Purpose |
|---------|-----|---------|
| **Traefik** | `http://your-server:8080` | Reverse proxy dashboard |
| **Homepage** | `http://your-server:3000` | Service dashboard |
| **Uptime Kuma** | `http://your-server:3001` | Monitoring |
