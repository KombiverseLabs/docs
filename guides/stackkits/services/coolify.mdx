Coolify is an optional app platform at `coolify.<domain>`.

## Prerequisites

<Note>
  Use one app platform path per workload. Do not configure the same app in Coolify and Dokploy at the same time.
</Note>

## Steps

<Steps>
  <Step title="Open Coolify">
    Open `https://coolify.<domain>` from the Node Hub.
  </Step>
  <Step title="Complete first-run setup">
    Create the owner account and verify the server resource limits before adding workloads.
  </Step>
  <Step title="Deploy a small app first">
    Start with a low-risk app route to confirm DNS, TLS, and container networking.
  </Step>
</Steps>

## Verify

- Coolify can start a simple service.
- The route resolves through Traefik.
- Resource usage remains within the server budget.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Coolify and Dokploy both try to own the same route">
    Pick one app platform for that route and remove the duplicate configuration from the other platform.
  </Accordion>
</AccordionGroup>
