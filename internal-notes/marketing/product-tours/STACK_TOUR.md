```bash

  --server https://stack.kombisphere.io \
```

## Self-Hosting

kombify Stack ist **Open Source** (MIT + AGPL):

### Docker Compose

```yaml
services:
  kombistack:
    image: ghcr.io/kombify/stack:latest
    ports:
      - "5260:5260"   # API
      - "5261:5261"   # UI
      - "5263:5263"   # gRPC (Agents)
    volumes:
      - ./data:/app/data
    environment:
      - KOMBISTACK_DOMAIN=stack.home.local
```
