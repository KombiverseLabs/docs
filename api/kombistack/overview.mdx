# KombiStack API

REST API for managing infrastructure stacks.

## Base URL

```
http://localhost:5260/api/v1
```

## Authentication

```bash
# PocketBase auth
curl -X POST http://localhost:5260/api/collections/users/auth-with-password \
  -d '{"identity": "admin@example.com", "password": "password"}'

# Use token
curl -H "Authorization: Bearer $TOKEN" \
  http://localhost:5260/api/v1/stacks
```
