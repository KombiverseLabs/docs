# Choosing a deployment model

Choose between self-hosted, SaaS, or hybrid deployment.

## Self-hosted (Open core)

**Pros:**
- Full control over data
- No subscription costs
- Custom modifications
- Privacy-focused

**Cons:**
- Manage infrastructure
- Handle updates
- No enterprise features

## SaaS (Managed)

**Pros:**
- No infrastructure to manage
- Automatic updates
- Enterprise SSO
- Team collaboration

**Cons:**
- Subscription costs
- Data in cloud
- Less customization

## Hybrid

**Pros:**
- Core tools self-hosted
- SaaS management layer
- Best of both worlds

**Cons:**
- More complex setup
- Requires both deployments

## Recommendation

- **Individuals:** Self-hosted
- **Teams:** SaaS or Hybrid
- **Enterprise:** Hybrid or SaaS
