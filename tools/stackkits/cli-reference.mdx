# CLI reference

The StackKit CLI provides commands for managing infrastructure blueprints.

## Installation

```bash
# Install via Go
go install github.com/KombiverseLabs/StackKits/cmd/stackkit@latest

# Or download binary
curl -fsSL https://raw.githubusercontent.com/KombiverseLabs/StackKits/main/install.sh | sh
```

## Commands

### stackkit list

List available StackKits:

```bash
stackkit list
```

### stackkit info

View StackKit details:

```bash
stackkit info base-homelab
```

### stackkit validate

Validate configuration:

```bash
stackkit validate kombination.yaml
```

### stackkit apply

Deploy infrastructure:

```bash
stackkit apply kombination.yaml
```

### stackkit destroy

Destroy infrastructure:

```bash
stackkit destroy kombination.yaml
```
