Central API gateway providing unified entry point for all microservices.

## Features

- JWT authentication (Zitadel OIDC)
- API key management
- Rate limiting per endpoint
- gRPC support
- Declarative configuration

## Tech stack

- **Gateway:** Kong OSS
- **Auth:** Zitadel JWT
- **Config:** decK
