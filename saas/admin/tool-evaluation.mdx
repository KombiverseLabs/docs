Evaluate and catalog tools for the kombify Cloud ecosystem.

## Features

- GitHub repository analysis
- npm/PyPI package discovery
- Version tracking
- Changelog monitoring
- Category assignment

## Evaluation criteria

- **Functionality:** Does it solve a real problem?
- **Maturity:** Is it production-ready?
- **Maintenance:** Is it actively maintained?
- **Documentation:** Is it well-documented?
- **Community:** Does it have community support?
