# Authentication overview

Configure authentication for KombiSphere.

## Methods

- **PocketBase:** Built-in auth for self-hosted
- **Zitadel:** Enterprise OIDC for SaaS
- **SSH keys:** Server access
- **API keys:** Programmatic access
