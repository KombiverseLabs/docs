Companions are conversational AI assistants with persistent memory. This guide covers how to configure and personalize them.

## Available Companions

| Companion | Focus | Best for |
|-----------|-------|----------|
| **Smart Home** | IoT, Home Assistant | Controlling smart devices, automations |
| **Personal** | Daily life, reminders | Personal productivity, context-aware help |
| **Work** | Projects, teams | Work-related tasks, project management |

## Choosing a Companion

Each Companion is specialized for different use cases. You can switch between them at any time — each maintains its own memory and conversation history.

## Customizing behavior

Navigate to **AI Settings > Companions** to configure:

- **Personality** — Adjust the Companion's communication style
- **Memory preferences** — Control what the Companion remembers
- **Tool access** — Which infrastructure tools the Companion can use
- **Delegation rules** — When the Companion should involve a Manager

## Memory system

Companions use a 4-layer memory system to maintain context:

1. **Identity** — Your preferences, goals, and personality settings
2. **Semantic** — Facts and relationships extracted from conversations
3. **Episodic** — Past conversations and events
4. **Working** — Current conversation context

<Note>
  Memory is stored locally in self-hosted mode. In SaaS mode, memory is encrypted and isolated per user.
</Note>
