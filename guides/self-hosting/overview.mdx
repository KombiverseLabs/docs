kombify is designed to run entirely on your own hardware. This section covers the main deployment paths and what you need to know before getting started.

## Deployment options

<CardGroup cols={2}>
  <Card title="Docker Compose" icon="docker" href="/guides/self-hosting/docker-compose">
    The recommended starting point for most self-hosters. Get the full stack running with a single command.
  </Card>
  <Card title="Kubernetes" icon="dharmachakra" href="/guides/self-hosting/kubernetes">
    For production environments that need scaling, rolling updates, and high availability.
  </Card>
</CardGroup>

## Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| OS | Ubuntu 22.04+ | Ubuntu 24.04 LTS |
| CPU | 2 cores | 4+ cores |
| RAM | 4 GB | 8+ GB |
| Disk | 20 GB | 50+ GB SSD |
| Docker | 24+ | Latest stable |

## What you'll need

- A Linux machine (physical or VM) with root access
- Docker and Docker Compose installed
- A domain name (optional, but recommended for TLS)
- Basic familiarity with the command line
