# KombiSphere API

Unified API gateway for all KombiSphere services.

## Base URL

```
https://api.kombisphere.io/v1
```

## Authentication

```bash
# JWT token
curl -H "Authorization: Bearer $TOKEN" \
  https://api.kombisphere.io/v1/tools

# API key
curl -H "X-API-Key: $API_KEY" \
  https://api.kombisphere.io/v1/tools
```
