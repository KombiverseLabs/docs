## Getting help

### Before asking

1. **Search the docs** â€” Check if your question is answered in the documentation
2. **Search Discord** â€” Someone may have asked the same question
3. **Check GitHub issues** â€” Look for existing bug reports

### How to ask

<Steps>
  <Step title="Choose the right channel">
    - **#help** â€” General questions
    - **#troubleshooting** â€” Specific issues
    - **#bug-reports** â€” Confirmed bugs
  </Step>

  <Step title="Provide context">
    Include:
    - kombify version
    - Operating system
    - Relevant configuration
    - Error messages
    - What you've tried
  </Step>

  <Step title="Use code blocks">
    ```yaml
    # Format your code
    stack:
      name: my-homelab
    ```
  </Step>

  <Step title="Be patient">
    Community members help in their free time. Please be respectful and patient.
  </Step>
</Steps>

- Treat everyone with respect

- Help others when you can
- Share your knowledge

### Stay on topic
- Keep discussions relevant

- Avoid spam and self-promotion

### Follow the rules
