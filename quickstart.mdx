# Quick start

Get the entire KombiSphere ecosystem running locally in under 5 minutes.

## Prerequisites

<CardGroup cols={2}>
  <Card title="Docker" icon="docker">
    Docker Engine 24.0+ with Compose v2.20+
  </Card>
  <Card title="System resources" icon="microchip">
    4GB RAM, 10GB disk space
  </Card>
</CardGroup>

## Installation

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/KombiverseLabs/KombiStack
    cd KombiStack
    ```
  </Step>

  <Step title="Start the services">
    ```bash
    docker compose up -d
    ```
    
    This starts:
    - **KombiStack Core** (API + PocketBase) on port 5260
    - **KombiStack UI** (SvelteKit) on port 5261
    - **KombiSim** (optional) on port 5270
  </Step>

  <Step title="Access the dashboard">
    Open your browser to [http://localhost:5261](http://localhost:5261)
    
    Default credentials:
    - Email: `admin@kombistack.local`
    - Password: `admin123` (change immediately!)
  </Step>

  <Step title="Create your first stack">
    Use the wizard to define your infrastructure:
    
    1. Choose a StackKit (e.g., `base-homelab`)
    2. Configure your nodes (IP addresses, SSH credentials)
    3. Select services (Traefik, Dokploy, etc.)
    4. Review and deploy
  </Step>
</Steps>

## Verify installation

<CodeGroup>
```bash Health check
curl http://localhost:5260/api/v1/health
```

```bash List stacks
curl http://localhost:5260/api/v1/stacks
```

```bash Check PocketBase
curl http://localhost:5260/_/
```
</CodeGroup>

## Service ports

| Service | Port | Description |
|---------|------|-------------|
| Core API | 5260 | REST API + PocketBase Admin |
| Frontend | 5261 | SvelteKit Dashboard |
| gRPC | 5263 | Agent communication (mTLS) |
| KombiSim | 5270 | Simulation API (optional) |
| KombiSim UI | 5271 | Simulation Dashboard (optional) |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port conflicts">
    If ports are already in use, modify `docker-compose.yml`:
    
    ```yaml
    services:
      kombistack:
        ports:
          - "5260:5260"  # Change to 5360:5260 if needed
    ```
  </Accordion>

  <Accordion title="Permission errors">
    Ensure Docker has proper permissions:
    
    ```bash
    # Linux: Add user to docker group
    sudo usermod -aG docker $USER
    newgrp docker
    ```
  </Accordion>

  <Accordion title="Services not starting">
    Check logs for errors:
    
    ```bash
    docker compose logs kombistack
    docker compose logs app
    ```
  </Accordion>
</AccordionGroup>
