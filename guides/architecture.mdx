# Architecture

KombiSphere combines open-source tools with a SaaS management layer.

## Components

### Open core tools
- **KombiStack:** Control plane
- **KombiSim:** Simulation engine
- **StackKits:** Infrastructure blueprints

### SaaS platform
- **KombiSphere Cloud:** User portal
- **KombiSphere Admin:** Admin center
- **KombiSphere API:** API gateway

## Data flow

User → Cloud Portal → API Gateway → Tools
