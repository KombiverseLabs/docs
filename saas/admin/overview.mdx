# KombiSphere Admin

KombiSphere Admin is the **management platform** for tool evaluation, pattern management, and StackKit configuration. It provides the "catalog" of available tools and configurations for the KombiSphere ecosystem.

## Data model

### Tools collection

```typescript
interface Tool {
  id: string;
  name: string;
  description: string;
  category: string;
  
  // Source
  source_type: 'github' | 'npm' | 'pypi' | 'manual';
  source_url: string;
  
  // Metadata
  stars: number;
  forks: number;
  issues: number;
  last_updated: string;
  
  // Versions
  latest_version: string;
  versions: Version[];
  
  // Status
  status: 'active' | 'deprecated' | 'archived';
  evaluation_status: 'pending' | 'approved' | 'rejected';
}
```

### Patterns collection

```typescript
interface Pattern {
  id: string;
  name: string;
  description: string;
  
  // Configuration
  tools: string[];  // Tool IDs
  architecture: string;
  
  // Metadata
  use_cases: string[];
  complexity: 'simple' | 'moderate' | 'complex';
  
  // Status
  status: 'draft' | 'published' | 'archived';
}
```

### StackKits collection

```typescript
interface StackKit {
  id: string;
  name: string;
  description: string;
  
  // Configuration
  tools: string[];  // Tool IDs
  patterns: string[];  // Pattern IDs
  
  // Metadata
  min_nodes: number;
  max_nodes: number;
  platforms: string[];
  
  // Status
  status: 'available' | 'schema_only' | 'deprecated';
}
```

## Discovery engine

### GitHub crawler

```go
type GitHubCrawler struct {
    client *github.Client
    db     *pocketbase.PocketBase
}

func (c *GitHubCrawler) Discover() error {
    // Search for homelab tools
    repos, err := c.client.Search.Repositories(context.Background(), 
        "homelab self-hosted", nil)
    
    for _, repo := range repos.Repositories {
        tool := &Tool{
            Name:        repo.GetName(),
            Description: repo.GetDescription(),
            SourceURL:   repo.GetHTMLURL(),
            Stars:       repo.GetStargazersCount(),
            Forks:       repo.GetForksCount(),
        }
        
        // Save to database
        c.db.Create("tools", tool)
    }
    
    return nil
}
```

### Version monitoring

```go
func (c *GitHubCrawler) MonitorVersions() error {
    tools, err := c.db.FindAll("tools")
    
    for _, tool := range tools {
        // Get latest release
        release, err := c.client.Repositories.GetLatestRelease(
            context.Background(), tool.Owner, tool.Repo)
        
        if release.GetTagName() != tool.LatestVersion {
            // New version detected
            c.notifyNewVersion(tool, release)
        }
    }
    
    return nil
}
```

## Authentication

KombiSphere Admin integrates with KombiStack for centralized user management:

```yaml
# Configuration
auth:
  enabled: true
  kombistack_url: http://kombistack:5260
  
  # Admin users
  admins:
    - admin@kombisphere.io
    - ops@kombisphere.io
```

See [KombiStack Auth Integration](https://github.com/KombiverseLabs/KombiSphere-Admin/blob/main/docs/KOMBISTACK_AUTH_INTEGRATION.md) for details.
