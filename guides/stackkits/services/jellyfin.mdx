Jellyfin is the media server exposed at `media.<domain>` when enabled.

## Prerequisites

<Note>
  Confirm your media path and storage budget before importing a large library.
</Note>

## Steps

<Steps>
  <Step title="Open Jellyfin">
    Open `https://media.<domain>` from the Node Hub.
  </Step>
  <Step title="Create the admin user">
    Complete Jellyfin's first-run setup and choose libraries that match the mounted media path.
  </Step>
  <Step title="Add users and clients">
    Add household users and connect TV, mobile, or browser clients after the library scan completes.
  </Step>
</Steps>

## Verify

- The library path is visible inside Jellyfin.
- Playback works on a browser client.
- Storage growth matches the server capacity.

<AccordionGroup>
  <Accordion title="No media appears">

  </Accordion>
