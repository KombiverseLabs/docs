Deploy the complete kombify Cloud stack using Docker Compose.

## Quick start

```bash
# Clone kombify-TechStack
git clone https://github.com/kombify/techstack
cd techstack

# Start services
docker compose up -d

# Access
# kombify TechStack: http://localhost:5261
# kombify Simulate: http://localhost:5271
```

## Services

- kombify TechStack Core (API + PocketBase)
- kombify TechStack UI
- kombify Simulate API
- kombify Simulate UI
- Headscale (optional)
